# 군지원 플래너 Next.js 웹앱

이 폴더는 Firebase Studio와 웹 배포 흐름에 맞추기 위한 Next.js 버전 골격입니다.

## 실행

```bash
npm install
npm run dev
```

기본 주소는 `http://localhost:3000` 입니다.

## 목적

- Firebase Studio에서 다루기 쉬운 App Router 기반 웹앱 제공
- Expo 앱에서 정리한 IA와 카피를 웹앱 구조로 재사용
- 이후 Firebase Auth, Firestore, Cloud Messaging 연동의 기반으로 사용
