# MVP 화면별 개발 체크리스트

## 첫 진입
- 비공식 서비스 고지
- 공식 사이트 재확인 문구
- 온보딩 시작 버튼

## 온보딩
- 필수 입력 필드 구성
- 우선 기준 선택
- 결과 화면 연결

## 홈
- 상태 요약 카드
- 다음 마감
- 오늘 할 일
- 최근 공지

## 추천
- 조건 입력 수정
- 결과 목록
- 상태 배지
- 특기 상세 이동

## 일정
- 리스트 보기
- 마감 임박 필터
- 추가모집 전용 필터

## 군별 가이드
- 핵심 개념 카드
- 군별 설명 템플릿

## 마이
- 프로필
- 저장 특기
- 알림 설정
- 체크리스트
