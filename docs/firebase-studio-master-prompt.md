# Firebase Studio용 마스터 생성 프롬프트

군지원 플래너라는 비공식 정보 서비스를 만든다.

핵심 목적:
- 사용자가 어떤 방식으로 군대에 갈 수 있는지 이해하게 한다.
- 지금 지원 가능한 분야를 조건 기반으로 바로 보여준다.
- 무엇을 더 준비해야 하는지 체크리스트로 알려준다.
- 언제까지 해야 하는지 일정과 역산 로직으로 보여준다.

필수 화면:
- 첫 진입
- 온보딩
- 홈
- 추천
- 일정
- 군별 가이드
- 마이

기술 요구:
- TypeScript
- Firebase Auth
- Firestore
- Cloud Functions 또는 Cloud Run
- Cloud Scheduler
- FCM
- PWA 대응 가능한 구조

데이터 신뢰도 배지:
- 공식 API 기반
- 공식 공지 기반
- 계산 추정치
- 운영자 검수 완료

제품 원칙:
- 예쁨보다 실용성
- 설명보다 행동 유도
- 긴 글보다 체크리스트와 상태 표시
- 최종 판단 전 공식 사이트 재확인 유도
